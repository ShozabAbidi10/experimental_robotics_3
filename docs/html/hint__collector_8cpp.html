Detailed Description

This files contains code for the 'hint_collector' node.

Author
Shozab Abidi
Version
1.0
Date
19/05/2022

Services:
/hint_loader_service /oracle_hint

Advertised Services:
/request_hint_collector

Description : This node provides the '/request_hint_collector' service which collects and store the hints one by one from the '/oracle_hint' service. Once the hint is collected, it checks if it is consistent and load them in the 'ARMOR' ontology knowlegde base (KB). Then it starts the ontology reasoner, and check if there is a complete and correct hypothesis. If the hints are inconsistent (malformed), or the hypothesis is incomplete or incorrect then the node return 'false' otherwise it return 'true'.
