def follow_marker_service.callback_follow_marker (   msg)

This is a callback function of 'request_follow_marker' service.

Returns
FollowMarkerResponse returns the scaned marker id to simple_action client node.

The function implements the exploration behavior of the robot in which it detects aruco markers in its surronding. Once the marker is detected then through visual servoing it move closer to it in order to better scan it.

◆ done()

def follow_marker_service.done ( )

This function reinstate robot's orientation and position for future exploration routine requests.

Returns
Bool

The function first rotates the robot anti-clock wise for one second and then stop both its linear and angular velocities. This is done so that in future request robot isn't facing towards the same aruco marker that it was before and start looking for new markers in the surrounding.
