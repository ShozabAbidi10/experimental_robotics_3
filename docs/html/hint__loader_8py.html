This node waits for 'hint_loader_service' service requests from the 'hintcollector' node. Based on the request recieved, it loads the hint in the ARMOR reasonser, start the reasoner to deduced a hypotheses based on the previously loaded hints and request ARMOR reasoner for the list of 'COMPLETE' hypotheses. If the recently deduced hypothesis is 'COMPLETE' then it checks its 'CORRECTNESS' by using the service '/oracle_solution'.
